BookTracker-программа для отслеживания прочитанных книг

функционал:
1.Добавление книг и заполнение информации о них
2.Просмотр информации о всех добавленных книгах
3.Средняя оценка по всем книгам
4.Количество книг от каждого автора
5.Удаление книг по названию и автору
6.Выход из приложения
7.Авто-сохранение при работе с данными,данные о книгах сохраняются в файл books.json

(для запуска нужен Python 3)запустите главный файл через консоль или двойным кликом:
python BookTrack.py

пример работы:
1.Добавить книгу
2.Показать все книги
3.Показать среднюю оценку
4.Статистика по авторам
5.Удалить книгу
6.Выход

книга 1984;автор Джордж Оруэлл;дата прочтения 12/25;оценка 5
